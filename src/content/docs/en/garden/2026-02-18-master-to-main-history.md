1.  **Trigger**: The **BLM (Black Lives Matter)** movement of 2020.
2.  **Core Controversy**: Although `master` in Git originally refers to "Master Copy," the long-standing use of **Master/Slave** terminology in computing inevitably evokes historical trauma related to slavery.
3.  **Industry Response**: GitHub CEO Nat Friedman responded to community calls in June 2020, and by October, the default branch for new repositories was officially changed to `main`.
4.  **De-politicization Efforts**: The tech world followed with a terminology sanitization movement:
    *   `Whitelist/Blacklist` -> `Allowlist/Denylist`
    *   `Master/Slave` -> `Main/Replica`

This has become an irreversible **"Political Correctness Industry Standard."**
